# DEPLOYMENT.md — 阿里云上线指南

> 给"完全没部署过网站"的设计师准备的、一步步照着点的清单。
> 选定方案:**阿里云 OSS + CDN + `.cn`/`.com.cn` 域名 + ICP 备案**。
> 总体周期:**域名注册 1 天 + 备案 2-3 周 + 上线 1 小时**。

---

## 0. 这套方案长什么样

```
用户浏览器
    ↓
阿里云 CDN (全国就近节点,加速)
    ↓
阿里云 OSS (对象存储,放 HTML/图片/视频/字体)
    ↑
你: 把 ~/Desktop/iOS_SnapTranslate_website 整个文件夹传上去
```

**为什么选 OSS + CDN 而不是云服务器**:
- OSS 是"按流量收费"的存储,几乎不要钱(一年几块到十几块)。
- 不用管 Linux、Nginx 配置,纯静态站就该这样部署。
- CDN 国内 200+ 节点,首屏比直连 OSS 快 3-5 倍。

**总成本预估(个人小流量)**:
- 域名 `.cn`:首年 ¥29 左右,续费 ¥35/年
- OSS 存储:1GB 以内 ¥0.12/月,流量按 GB 计费,一年几块钱
- CDN:免费额度每月 10GB,小流量基本免费
- HTTPS 证书:免费(阿里云提供单域名免费证书)

## 1. 注册阿里云账号 + 个人实名

> 这一步只能你来,我帮不了——涉及你的身份证和手机号。

1. 打开 https://www.aliyun.com → 右上角「免费注册」
2. 用手机号注册,设置密码
3. **登录后,右上角头像 → 「实名认证」**
4. 选「个人实名」,上传身份证正反面 + 人脸识别(支付宝扫脸最快)
5. 等待几分钟审核通过

**✅ 完成标志**:右上角头像点开能看到「个人实名」绿色标志。

## 2. 买域名

### 2.1 注册域名

1. 进入 https://wanwang.aliyun.com (阿里云域名)
2. 顶部搜索框输入你想要的域名(不带后缀),回车
3. 看可用结果,选 `.cn` 或 `.com.cn`,点「加入清单」→「立即结算」

### 2.2 选哪个域名后缀

| 后缀 | 价格 | 备案难度 | 备注 |
|---|---|---|---|
| `.cn` | ¥29/年(首年) | 个人可以备案 | **推荐**,短、便宜、国内常用 |
| `.com.cn` | ¥29/年 | 个人可以备案 | 商业感稍强 |
| `.com` | ¥55/年 | 个人可以备案 | 国际通用,但解析到国内仍要备案 |

**建议**:`.cn`。除非你已经有一个 `.com` 心头好。

### 2.3 命名建议

- 主产品叫"快捷识屏",可以考虑:`kuaijieshipping.cn` / `kjsp.cn` / `snaptranslate.cn`
- 如果 4-6 字母短域名都没了,可以加描述词,比如 `snaptrans-app.cn`

### 2.4 域名实名认证(必须!)

域名买完后,**域名本身也要实名**(和阿里云账号实名是两回事)。

1. 控制台搜「域名」→ 进入域名列表
2. 找到刚买的域名 → 「管理」→ 左侧「域名信息」
3. 看「实名认证状态」:如果是「未认证」,点旁边的「立即认证」
4. 选「个人」→ 关联你的支付宝/淘宝账号(身份信息会自动带入)→ 提交
5. 等 1-3 个工作日审核

**✅ 完成标志**:域名信息里「实名认证状态」变为「已认证」。

> 注意:**域名没实名认证之前,无法发起备案**。

## 3. 发起 ICP 备案

### 3.1 准备工作

需要这些东西:
- 阿里云账号(已实名,见第 1 章)
- 域名(已实名,见第 2 章)
- 身份证正反面照片(清晰)
- 一张你举着身份证的自拍(用阿里云 App 现场拍)
- **必须有一个备案"资源"** —— 也就是 OSS 或者云服务器。所以备案前要先买 OSS(见第 4.1 步)

**几个填写要点**:
- ⚠️ **不要写"翻译"、"识图"、"AI"这类词在网站名称里** —— 容易被管局要求补充材料(认为涉及在线服务)。直接写产品名"快捷识屏"最安全。
- ⚠️ **服务内容选「博客/个人空间」最稳**。选「电子商务」或「在线工具」会让审核变严。
- ⚠️ **网站内容描述明确写"纯静态展示"、"不收集用户信息"** —— 这两个表述让管局放心。

### 3.3 发起备案流程

1. 控制台 → 顶部搜索「备案」→ 进入「ICP 备案」
2. 选「我有一个备案需求」→ 选「新增网站」(首次备案)
3. 验证备案类型:个人 → 上传身份证 → 人脸核验(用阿里云 App 扫码)
4. 选「备案接入商」:阿里云
5. 选「关联的产品」:**这里要选 OSS 或服务器,所以必须先买 OSS!** 见第 4.1 章
6. 填上面模板的所有信息 → 提交
7. 等阿里云初审(1-2 工作日)→ 等管局终审(7-20 工作日)
8. 中间可能会接到管局电话/短信回访,问你"网站是不是你做的"、"内容是什么",照实回答即可。

**✅ 完成标志**:收到工信部短信,内容是"您的备案号为 [xx ICP 备 xxxxxxxx号]"。这个备案号要放到网站底部(我可以在备案通过后帮你加上)。

## 4. 买 OSS + 准备文件

### 4.1 开通 OSS(可以在发起备案前做,因为备案要绑定 OSS)

1. 控制台 → 搜「OSS」→ 进入「对象存储 OSS」
2. 第一次进会让你开通服务,**按量付费**(不要选包年包月,小流量按量便宜得多)
3. 开通后点左侧「Bucket 列表」→「创建 Bucket」
4. 关键参数:
   - **Bucket 名称**:`snaptranslate-website`(全网唯一,只能小写字母和数字)
   - **地域**:**华东1(杭州)** 或离你最近的(华北/华南)
   - **存储类型**:标准存储
   - **同城冗余存储**:关闭(省钱)
   - **版本控制**:关闭
   - **读写权限**:**公共读** ⚠️ 一定要选这个,不然外人访问不到
   - **服务端加密**:无
5. 创建完成

### 4.2 开启静态网站托管

1. 进入刚创建的 Bucket
2. 左侧「数据管理」→「静态页面」
3. 「设置」:
   - 默认首页:`index.html`
   - 默认 404 页:留空(或填 `index.html`)
4. 保存

### 4.3 上传网站文件

**方式 A:控制台拖拽**(简单,适合首次上传)
1. Bucket → 「文件管理」→ 「上传文件」
2. 把这些文件夹/文件**整个**拖进去(保持目录结构):
   - `index.html`
   - `privacy.html`
   - `tutorial.html`
   - `public/` (整个文件夹,含 fonts/images/videos)
3. 上传后访问 `https://[bucket名].oss-cn-hangzhou.aliyuncs.com/index.html` 应该能看到网站

**方式 B:命令行批量上传**(后续更新方便,我能帮你做)
- 安装 ossutil → 配置 AccessKey → `ossutil sync ./ oss://snaptranslate-website/ --exclude .git/* --exclude .DS_Store`
- 备案通过后再告诉 Claude,可以一条命令同步整个项目

## 5. 备案通过后:绑定域名 + 开 CDN

### 5.1 OSS 绑定自定义域名

1. OSS Bucket → 左侧「传输管理」→「域名管理」→「绑定域名」
2. 域名填:你买的域名(比如 `kuaijieshipping.cn`)
3. 「自动添加 CNAME 记录」勾上(阿里云会自动给你域名加解析)
4. 提交

### 5.2 申请免费 HTTPS 证书

1. 在「域名管理」的列表里,找到刚绑定的域名
2. 点「证书托管」→「免费证书」→ 申请
3. 等几分钟下发证书
4. 开启「强制 HTTPS」(把 http:// 自动跳 https://)

### 5.3 开 CDN 加速(可选但强烈推荐)

> 不开 CDN 也能用,但首屏会慢 2-3 倍。

1. 控制台 → 搜「CDN」→ 进入「内容分发 CDN」
2. 「添加域名」:
   - 加速域名:`你的域名`
   - 业务类型:**网页/小文件**
   - 源站类型:**OSS 域名** → 选刚才的 Bucket
   - 加速区域:**仅中国内地**(只服务国内用户,便宜)
3. 提交,等几分钟生效
4. CDN 也要绑定 HTTPS 证书(同样的流程)

### 5.4 网站底部加备案号

备案通过后,工信部要求:**网站底部必须显示备案号 + 可点击跳转工信部**。

我可以在你拿到备案号后帮你改 3 个 HTML 的 footer,加上类似:
```html
<a href="https://beian.miit.gov.cn/" target="_blank">[备案号]</a>
```

## 6. 常见坑

| 坑 | 解决 |
|---|---|
| 备案被驳回,理由"网站内容描述模糊" | 按 3.2 模板重填,强调"静态展示、不收集信息" |
| 备案被驳回,理由"网站名称涉及行业资质" | 改成"快捷识屏"这种纯品牌名,别带"翻译/工具" |
| OSS 默认域名访问 HTML 不渲染、直接下载 | 文件管理 → 文件属性 → 检查 Content-Type 是 `text/html` |
| 首页加载视频卡顿 | 视频太大(`public/videos/*.mp4` 各几 MB),建议压到 2MB 内 |
| logo 加载慢 | `public/images/logo2.png` 是 1.1MB,建议 tinypng 压一下 |
| 阿里云 OSS 流量贵 | 一定要开 CDN!CDN 回源到 OSS 走内网免费,用户流量走 CDN 便宜 |
