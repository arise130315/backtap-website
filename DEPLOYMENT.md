## 2. 域名:从"命名审核"到"可发起备案"

`.cn` 域名有 4 个状态,**必须全过**才能发起备案:

| 步骤 | 周期 | 你需要做什么 |
|---|---|---|
| ① 已购买 | 已完成 | - |
| ② **命名审核**(当前阶段) | 1-5 工作日 | 等。CNNIC 审核名字是否合规 |
| ③ 域名实名认证 | 1-3 工作日 | 提交身份证 |
| ④ 实名审核通过 | 自动 | 等 |

### 命名审核完成后,做实名认证

1. 控制台 → 顶部搜「域名注册」→「我的域名」
2. 找到 `backtap.cn` → 点「更多」→「域名信息修改/实名认证」
3. 域名持有者类型:**个人**
4. 域名持有者名称:**和身份证完全一致**(必须和你账号实名是同一个人)
5. 上传:身份证正面 + 反面 + 手持身份证半身照
6. 提交,等审核

**✅ 完成标志**:`backtap.cn` 状态显示「正常」+「已实名」。这时候才能去发起 ICP 备案。

> ⚠️ **命名审核期间不要做任何操作**(改信息、转入转出都会重置审核)。

## 3. 买 COS + 准备好"备案要绑的资源"

腾讯云规定:**发起备案必须绑定一个备案资源**(COS 或服务器或 CDN)。**这一步可以在域名实名通过之前提前做**,不必等。

### 3.1 开通 COS(对象存储)

1. 控制台 → 顶部搜「对象存储」→ 进入 COS 控制台
2. 第一次进会让你开通服务,点开通即可(免费开通,按量付费)
3. 左侧「存储桶列表」→ 「创建存储桶」
4. 关键参数:
   - **名称**:`backtap-website`(腾讯云会自动追加 APPID,最终是 `backtap-website-1300xxxxxx`)
   - **所属地域**:**广州** 或 **上海**(选离你最近的国内地域,影响访问速度)
   - **访问权限**:**公有读私有写** ⚠️ 必须选这个,不然外网访问 403
   - **请求域名**:默认即可(后续会绑你的 `backtap.cn`)
   - 其余保持默认
5. 创建完成。记住这个存储桶名称(包含 APPID 的完整名),后面备案要用。

### 3.2 开启静态网站托管

1. 进入刚创建的存储桶
2. 左侧「基础配置」→「静态网站」→ 「编辑」
3. 关键参数:
   - **状态**:开启
   - **索引文档**:`index.html`
   - **错误文档**:留空(或填 `index.html` —— 让所有 404 都回首页)
4. 保存。
5. 保存后会显示一个「访问节点」URL,类似 `backtap-website-1300xxx.cos-website.ap-guangzhou.myqcloud.com`。**这就是临时预览地址**,备案没通过期间用它看效果。

### 3.3 上传网站文件

**方式 A — 控制台拖拽**(简单,适合首次):

1. 存储桶 → 「文件列表」→ 「上传文件」/「上传文件夹」
2. **依次上传这些**(保持目录结构!):
   - 文件:`index.html`、`privacy.html`、`tutorial.html`
   - 文件夹:`public/`(整个文件夹,含 fonts/images/videos)
3. 上传时:勾选「保留目录」、文件权限「继承存储桶」
4. 上传完打开 3.2 提到的「访问节点」URL,应该能看到完整网站

**方式 B — 命令行同步**(推荐,以后更新方便,Claude 可以帮做):
- 工具:`coscli`(腾讯云官方命令行)
- Claude 可以帮你装好 coscli、配置 SecretKey,然后一条命令同步整个项目:
  ```bash
  coscli sync ./ cos://backtap-website-1300xxx/ --exclude ".git/*" --exclude "*.DS_Store" --exclude "*.md"
  ```
- 等你 COS 建好之后告诉我,我陪你配。

### 3.4 验证(必做)

打开 3.2 的「访问节点」URL,逐个检查:
- [ ] `首页` 能打开,字体正常(Inter 数字 + 阿里巴巴普惠体中文)
- [ ] 视频能播放(`快捷分析.mp4` / `快捷翻译.mp4`)
- [ ] 点击「使用教程」、「隐私政策」能正常跳转
- [ ] 移动端汉堡菜单能展开

如果哪里没显示对,告诉我我来排查。

## 4. 发起 ICP 备案

### 4.1 准备工作

需要这些东西:
- ✅ 腾讯云账号(已实名)
- ✅ `backtap.cn` 域名(已实名通过)
- ✅ COS 存储桶(用作"备案资源")
- 身份证正反面照片(清晰、四角完整、无反光)
- 上传过程中会让你下载「腾讯云 App」做人脸核验,提前装好

### 4.2 网站信息填写模板(直接抄)

下面这份给你照填,审核通过率很高(避免敏感词、过度承诺):

```

│ 投资人/法人:[你的名字]                           │
│                                                  │

├─────────────────────────────────────────────────┤
│ 负责人姓名:[你的名字]                            │
│ 证件类型:身份证                                  │
│ 证件号码:[同上]                                  │
│ 办公电话:[填手机号]                              │
│ 手机号码:[手机号]                                │
│ 邮箱:arise130315@gmail.com                       │
│ 应急电话:[手机号]                                │
│                                                  │
│ 网站信息                                         │
├─────────────────────────────────────────────────┤
│ 网站名称:快捷识屏                                │

│ 域名:backtap.cn                                  │
│ 服务内容:博客/个人空间                           │
│   (如果选项里没有"博客",选"其他"→ 描述写"个人  │
│   开发者作品展示")                              │
│ 网站语言:中文简体                                │
│ 是否涉及前置审批:否                              │
│ 网站内容描述:                                    │
│   本网站为个人开发者作品 iOS 工具应用"快捷识屏" │

│   入口。网站为纯静态展示页面,不提供用户注册、   │
│   评论、内容发布等交互功能,不收集用户个人信息。 │
│   网站负责人为独立开发者本人,自有著作权。       │
└─────────────────────────────────────────────────┘
```

**几个填写要点**:
- ⚠️ **网站名称建议写"快捷识屏"** —— 不要带"翻译/识图/AI/工具"这类词。这些词会触发更严格的审核,有可能被要求补充行业资质。
- ⚠️ **服务内容选「博客/个人空间」最稳**。选「电子商务」或「在线工具」会大大延长审核时间。
- ⚠️ **网站内容描述必须明确写"纯静态展示"、"不收集用户信息"** —— 这两个表述让管局放心。
- ⚠️ **不要在描述里提"翻译服务"、"OCR"、"AI 识图"** —— 描述网站行为(展示介绍),不要描述产品功能。

### 4.3 发起备案流程

1. 控制台 → 顶部搜「备案」→ 进入「网站备案」
2. 「新增备案」→ 选「我有主体没有网站」(首次)或「我没主体也没网站」(都是新的)
3. 验证手机号 + 关联 `backtap.cn`
4. 上传身份证 + 用「腾讯云 App」扫码人脸核验
5. 选「备案接入商」:腾讯云
6. 选「关联的腾讯云资源」:刚才创建的 COS 存储桶(系统会自动列出来)
7. 按 4.2 模板填所有信息 → 提交
8. **腾讯云初审**(1-2 工作日,通常 24 小时内)
9. **管局终审**(7-20 工作日)
10. 中间可能会接到管局电话/短信回访,问"网站是不是你做的"、"内容是什么"、"是否涉及收集用户信息" —— 照实回答(纯静态展示页 / 不收集用户信息 / 个人独立开发者)

**✅ 完成标志**:收到工信部短信,内容是「您的备案号为 [xxICP备xxxxxxxx号]」。

## 5. 备案通过后:绑定域名 + 开 CDN + 加备案号

### 5.1 给 COS 绑定 `backtap.cn`

1. COS 存储桶 → 「域名与传输管理」→「自定义源站域名」→「添加域名」
2. 域名填:`backtap.cn`(主域名)和 `www.backtap.cn`(各加一次)
3. **是否开启 CDN 加速**:✅ 开启(强烈推荐)
4. **HTTPS 证书**:选「自动申请免费证书」→ 腾讯云会自动签发 TrustAsia DV 证书
5. 提交。腾讯云会自动给 DNSPod 加 CNAME 记录(因为域名也在腾讯云,这一步自动完成)

### 5.2 强制 HTTPS

1. 控制台 → CDN → 域名管理 → `backtap.cn`
2. 「HTTPS 配置」→ 开启「强制跳转 HTTPS」(让所有 http:// 自动跳 https://)
3. 「访问控制」可保持默认

### 5.3 网站底部加备案号(管局要求)

备案通过后,**网站底部必须显示备案号 + 可点击跳转工信部**,否则可能被注销备案。

  粤ICP备xxxxxxxx号-1
</a>

## 6. 常见坑

| 坑 | 解决 |
|---|---|
| 命名审核被驳回(`.cn` 特有) | 名字里别有数字开头、敏感词、英文+中文混合。`backtap` 是纯英文单词,应该没问题 |
| 域名实名照片被驳"反光/模糊" | 用手机拍,光线均匀,身份证四角必须在画面内 |
| 备案被驳"内容描述与服务不匹配" | 按 4.2 模板抄,**不要自己发挥**写得太详细 |
| 备案被驳"网站名称涉及行业资质" | 改成纯品牌名"快捷识屏",别带"翻译/工具/AI" |
| COS 访问 HTML 不渲染、直接下载 | 文件 → 详情 → 检查 Content-Type 是 `text/html`(`.html` 后缀通常自动识别) |
| 视频卡顿、流量费用高 | `public/videos/*.mp4` 各几 MB,建议用 HandBrake 压到 2MB 内。Claude 也能帮压 |
| logo 加载慢 | `public/images/logo2.png` 是 1.1MB。建议用 tinypng.com 压到 100KB 内 |
| 流量费意外飙升 | 一定开 CDN!CDN 回源到 COS 走腾讯云内网免费 |
